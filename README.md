# Cardy's Tacos and Takeaways Website

Welcome to the official website project for **Cardy's Tacos and Takeaways**! This repository contains the source code and assets for Cardy's digital presence, designed to showcase the menu, location, and unique vibe of Cardy's Tacos and Takeaways.

## 🚀 Project Overview
This website is a modern, responsive, and visually appealing site for Cardy's Tacos and Takeaways. It features:
- A vibrant homepage introducing Cardy's brand
- Menu highlights with mouth-watering images
- Easy-to-find contact and location information
- Mobile-friendly design for customers on the go
- Fast loading and accessible layout

## 📁 Project Structure
```
index.html                # Main landing page
assets/
  css/
    styles.css            # Main stylesheet
    styles.locked.css     # Additional/locked styles
  images/                 # All images and graphics
  js/
    script.js             # Main JavaScript file
```

## 🛠️ Getting Started
1. **Clone the repository:**
   ```bash
   git clone <repo-url>
   ```
2. **Open the project folder:**
   Navigate to the project directory and open `index.html` in your browser.
3. **Customize:**
   - Update images in `assets/images/` as needed
   - Edit content in `index.html` for menu, hours, and specials
   - Adjust styles in `assets/css/styles.css` for branding

## 🌮 Features
- **Responsive Design:** Looks great on mobile, tablet, and desktop
- **Easy Navigation:** Simple, intuitive layout for all users
- **Delicious Visuals:** High-quality images to entice customers
- **Contact & Location:** Quick access to find and contact Cardy's
